Fake Chromecast

Open with ?fakeChromecast=available to test Chromecast plugin logic without a device.

Use ?fakeChromecast=no-devices or ?fakeChromecast=available&fakeChromecastLoad=fail for failure paths.

Config examples

Shorthand (default receiver): chromecast: true

Explicit config:

chromecast: {
  appId: 'CC1AD845',            // default media receiver
  autoJoinPolicy: 'tab_and_origin_scoped',
}
